# 💳 Finance Anomaly Detection using Machine Learning

AI-based financial anomaly detection system using XGBoost and LightGBM to identify fraudulent transactions.

---

## 📌 Project Overview

This project predicts fraudulent credit card transactions using Machine Learning algorithms.

Features:
- Fraud prediction
- User/Admin modules
- Accuracy comparison
- User records management
- Visualization graphs

---

## 🛠️ Technologies Used

- Python
- Django
- Machine Learning
- SQL
- Pandas
- NumPy
- Scikit-learn
- HTML
- CSS

---

## ⚙️ Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

Run project:

```bash
python manage.py runserver
```

Open browser:

```bash
http://127.0.0.1:8000/
```
